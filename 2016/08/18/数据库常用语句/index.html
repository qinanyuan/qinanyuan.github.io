数据库常用语句

  • oracle

记录一些数据库常用命令语句，再也不用因为sql语句发愁了！

Oracle之修改表中字段sql

添加字段

alter table tablename add (column datatype [default value][null/not null],….);

修改字段

alter table tablename modify (column datatype [default value][null/not null],….);

删除字段

alter table tablename drop (column);

添加、修改、删除多列的话，用逗号隔开。

重命名表

ALTER TABLE table_name RENAME TO new_table_name;

修改列的名称

ALTER TABLE table_name RENAME COLUMN supplier_name to sname;

修改列的数据类型并且保留原值

------ 先将该字段换一个列名
ALTER TABLE MARKETING_APPLICATION RENAME COLUMN TARGET_NAME TO bak_TARGET_NAME;  
------新增一个与原列名相同的列  
ALTER TABLE MARKETING_APPLICATION ADD TARGET_NAME CLOB; 
------将修改后的列的值赋值给新增的列
UPDATE MARKETING_APPLICATION SET TARGET_NAME = bak_TARGET_NAME;
------删除修改后的列
ALTER TABLE MARKETING_APPLICATION DROP COLUMN bak_TARGET_NAME;

这个是在修改主键类型的时候很有用处
